# 이분 그래프

## 이분 그래프란?

이분 그래프(bitpartie graph)란 인접한 정점끼리 서로 다른 색으로 칠해서 모든 정점을 두 가지 색으로만 칠할 수 있는 그래프를 의미한다. 

## 이분 그래프 판별
이분 그래프인지 확인하는 방법에는 BFS와 DFS 탐색을 이용하면 된다. 서로 인전한 정점이 같은 색이면 이분 그래프가 아니다.

 1. BFS, DFS 탐색을 진행하며 정점을 방문할 때 두 가지 색 중 하나를 칠한다.
 2. 다음 정점을 방문하며 자신과 다른 색으로 칠한다.
 3. 만약 인접한 정점의 색이 자신과 동일하면 이분 그래프가 아니다.

## BFS 활용

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;
import java.util.Scanner;

public class BOJ1707 {

    private static List<List<Integer>> graph;
    private static int[] team;
    private static boolean isParticiple;

    private static final int RED = 1;

    private static void bfs(int v) {
        Queue<Integer> queue = new LinkedList<>();
        queue.add(v);
        team[v] = RED;

        while (!queue.isEmpty()) {
            Integer node = queue.poll();
            // 인접 노드 탐색
            for (Integer i : graph.get(node)) {
                // 이전 팀과 같은 경우
                if (team[node] == team[i]) {
                    isParticiple = false;
                    return;
                }

                if (team[i] == 0) {
                    team[i] = team[node] * -1;
                    queue.add(i);
                }
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int t = scanner.nextInt();

        while (t-- > 0) {
            int v = scanner.nextInt();
            int e = scanner.nextInt();

            graph = new ArrayList<>();
            for (int i = 0; i <= v; i++) {
                graph.add(new ArrayList<>());
            }

            for (int i = 0; i < e; i++) {
                int a = scanner.nextInt();
                int b = scanner.nextInt();

                graph.get(a).add(b);
                graph.get(b).add(a);
            }

            team = new int[v + 1];
            isParticiple = true;
            for (int i = 1; i <= v; i++) {
                if (team[i] == 0) {
                    bfs(i);
                    if (!isParticiple) {
                        break;
                    }
                }
            }

            if (isParticiple) {
                System.out.println("YES");
            } else {
                System.out.println("NO");
            }
        }
    }
}
```

## DFS 활용

```java
import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class BOJ1707 {

    private static List<List<Integer>> graph;
    private static int[] team;
    private static boolean isParticiple;

    private static final int RED = 1;

    private static void dfs(int v, int color) {
        team[v] = color;

        for (Integer i : graph.get(v)) {
            if (team[v] == team[i]) {
                isParticiple = false;
                return;
            }

            if (team[i] == 0) {
                dfs(i, color * -1);
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int t = scanner.nextInt();

        while (t-- > 0) {
            int v = scanner.nextInt();
            int e = scanner.nextInt();

            graph = new ArrayList<>();
            for (int i = 0; i <= v; i++) {
                graph.add(new ArrayList<>());
            }

            for (int i = 0; i < e; i++) {
                int a = scanner.nextInt();
                int b = scanner.nextInt();

                graph.get(a).add(b);
                graph.get(b).add(a);
            }

            team = new int[v + 1];
            isParticiple = true;
            for (int i = 1; i <= v; i++) {
                if (team[i] == 0) {
                    dfs(i, RED);
                    if (!isParticiple) {
                        break;
                    }
                }
            }

            if (isParticiple) {
                System.out.println("YES");
            } else {
                System.out.println("NO");
            }
        }
    }
}
```
