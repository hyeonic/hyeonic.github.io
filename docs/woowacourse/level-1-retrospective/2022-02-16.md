## 목표

우아한테크코스를 진행할 때 하루를 되새기며 마무리하기 위해 회고를 작성한다.

## 데일리 미팅

블로그 전도사 컨셉으로 데일리 미팅의 리드를 맡아 진행하였다. 블로그를 시작하게 된 계기와 장점들에 대해 나누었고, 실제 경험을 바탕으로 다양한 블로그 플랫폼을 추천하였다. 

매번 유의미한 자료를 남기기위해 메타인지(?)를 지속적으로 시도하지만 아직 부족한 점이 많다고 생각한다. 또한 현재 통계 관련된 설정이 부족하기 때문에 작성한 게시글에 대한 객관성이 부족하다. 즉 현재 필자가 어떤 부분이 문제인지 모른체 똥글을 양산해나가고 있을지도 모른다.

올해 우테코를 진행하며 블로그에 대한 방향성이나 플랫폼을 필자의 개성으로 채워갈 수 있도록 많은 고민을 해봐야 할 것 같다.

내일 데일리 미팅은 브리 없이 진행된다. 남은 크루들 끼리 대화를 잘 나눌 수 있을지 걱정과 설렘이 공존한다.

## git 특강

git 관련 특강을 진행하였다. 작년 팀프로젝트를 하며 고생한 탓에 git 사용에 있어서 큰 어려움은 없었다. 다만 아직 git 기반 지식들은 부족하기 때문에 확실하게 이해하고 사용한다고는 생각하지 않는다.

간단한 키워드 및 구성 요소에 대한 수업이 진행되고 랜덤으로 3명의 크루가 모여 간단한 미션을 진행했다.

[git-mission](https://github.com/hyeonic/git-mission)<br>

이번에 미션을 함께한 크루는 릭과 소주캉이다. 다들 처음보는 크루들이라 어색했지만 간단히 인사를 나누고 서둘러 미션을 진행했다.

미션 난이도는 크게 어렵지 않았다. 제한 시간안에 의도한 대로 모두 동작하였고 회고까지 마무리할 수 있었다. 확실히 git은 직접 실습하여 맞아봐야 저절로 느는 것 같다. 1년 전만해도 commit, push만 해도 후덜덜했는데 이제는 당연하게 느껴진다. 이런 경험을 통해 안아프게 맞는법을 열심히 습득해야 겠다.

.git 내부 구조를 학습하기 위한 좋은 글을 찾았다. [.git 내부 구조 파헤치기](https://tecoble.techcourse.co.kr/post/2021-07-08-dot-git/)

주말간 수업과 함께 간단히 정리해보면 좋을 것 같다.

## 보이는 라디오 강평

보이는 라디오 강평을 진행했다. 모든 팀들이 자신들의 킬링파트를 감추기 위해 짧게 진행했다. 하지만 그 짧은 티저조차 너무 재밌었다. 무튼 내일 기대가 된다.

## 생각 정리

오늘 원시값 포장, VO에 대한 자료를 하루종일 찾은 거 같다. 이전에 단순히 안다고 생각하고 넘어간 탓에 이제서야 조금은 이해가 간다. 이런 기회를 통해 메타인지에 대한 중요성을 더욱 느끼는 것 같다.

이렇게 하나, 둘 씩 놓쳤던 지식들을 채워가는 시간이 좋다. 이런 시간이 없었다면 부족한 지식들로 뒤꿈치가 까진채 앞으로 달려가고 있었을 것이다. 미션을 진행하는 지금은 반창고를 아주 든든하게 챙기고 달려가는 기분이다.
