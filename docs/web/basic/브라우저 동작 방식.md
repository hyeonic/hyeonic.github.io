# 브라우저 동작 방식

## 요약

 1. 주소창에 url을 입력하고 Enter를 누르면 서버에 요청이 전송된다.
 2. 해당 페이지에 존재하는 여러 자원들이 보내진다.
 3. 브라우저는 해당 자원이 담긴 html, csss를 W3C 명세에 따라 렌더링 엔진이 해석을 진행한다.
 4. 렌더링 엔진은 html 파싱 과정을 우선적으로 시작한다. html 파서가 문서에 존재하는 어휘와 구문을 분석하여 DOM 트리를 구축한다.
 5. CSS 파싱 과정을 시작한다. CSS 파서가 모든 CSS 정보를 스타일 구조체로 생성한다.
 6. 이 2가지를 연결하여 렌더 트리를 만든다. 렌더 트리를 통해 문서가 시각적 요소를 포함한 형태로 구성된 상태이다.
 7. 화면에 배치를 시작하고 UI 백엔드가 노드를 돌며 형상을 그린다.
 8. 빠른 브라우저 표시를 위해 자원을 전송 받으며 동시에 화면에 표시한다.
